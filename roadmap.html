Submission Requirements

Participants must submit a presentation, driven by 1 clear spatial concept, presented in 3 parts:

Clear Visual Exposé: up to 9 renderings showcasing the design vision—i.e. 3 for each phase.
Technical Exposé: plans, section, system diagrams detailing the evolving spatial proposal.
Brief Written Exposé: a 600 word overview, elucidating the core concepts underpinning the final design, explaining the planning process, and the growth strategy, and introducing the KPIs.

Submission Format

A combined PDF file of 3 landscape A0 posters:
- At printable resolution 200-300 DPI, with all images and links embedded.
- Written English text and statistics must be integrated in the design of the posters.
- Using an open type font (OTF) such as Helvetica Neue, at point 13 for the body copy.
- It is encouraged to fully cover the background for each A0 poster, for instance with a full spread rendering, with text, diagrams and additional images superimposed on it.
A 2 minutes videopresenting the scheme with English voiceover: 1920x1080 pxl (MP4, 1080p).
Entries are anonymousand are only marked by the provided entry code that participants receive on registration. This code becomes the file name of the PDF and MP4 that are submitted.
